# 🌈 Clear Spectrum – Autism Support Mobile Application

*A Human-Computer Interaction (HCI) UX/UI Case Study*

Clear Spectrum is a mobile application concept designed to support autistic adults through a calm, accessible, and sensory-friendly digital experience. Developed as a **Human-Computer Interaction (HCI) semester project**, the application focuses on reducing cognitive overload while helping users manage daily routines, emotions, communication, and caregiver collaboration.

Designed using **Figma**, the project follows **User-Centered Design (UCD)** principles and accessibility guidelines to create an inclusive interface that prioritizes simplicity, predictability, and ease of use.

---

# 📖 About the Project

Many existing productivity and wellness applications are designed for the general population and often contain cluttered interfaces, excessive animations, bright colors, and complex navigation. These design choices can become overwhelming for autistic individuals who may experience sensory sensitivities and cognitive overload.

**Clear Spectrum** addresses these challenges by providing a minimalist, predictable, and calming mobile experience. The application focuses on improving daily independence through structured routines, emotional support, simple communication tools, and caregiver collaboration while maintaining a sensory-friendly interface.

The project demonstrates the practical application of Human-Computer Interaction principles, accessibility standards, UX research, and modern UI design methodologies.

# 🌟 Key Features

## Visual Routine Scheduler

A structured routine planner designed to help users organize their daily activities using simple icons and visual timelines.

Features include:

* Daily schedules
* Task completion tracking
* Visual reminders
* Predictable layouts

---

## AAC Communication Board

An Augmentative and Alternative Communication (AAC) board that allows users to communicate their needs using visual icons instead of text-heavy interactions.

Features include:

* Emotion icons
* Object icons
* Action icons
* Quick communication
* Non-verbal interaction support

---

## Mood & Behavior Tracking

Allows users and caregivers to record emotions, moods, and behavioral patterns.

Features include:

* Mood logging
* Emotion tracking
* Behavioral observations
* Daily records

---

## Caregiver Dashboard

A dedicated workspace that enables caregivers to monitor user progress and identify behavioral trends.

Features include:

* Mood history
* Behavioral logs
* Daily observations
* User progress tracking

---

## Community Support

Provides a safe and minimal communication environment where users can connect with others.

Features include:

* Community discussions
* Safe communication
* Minimal interface distractions

---

## Calm User Interface

The interface is designed to reduce anxiety and cognitive overload through:

* Minimal layouts
* Simple navigation
* Soft color palettes
* Large touch targets
* Consistent interface patterns

# 🌗 Dual Theme Support

The application includes two carefully designed themes to accommodate different environments and sensory preferences.

## ☀️ Light Theme

Designed for daytime use with soft neutral backgrounds that reduce glare while maintaining excellent readability.

Features:

* Soft white backgrounds
* Muted color palette
* High readability
* Comfortable outdoor visibility

---

## 🌙 Dark Theme

Designed for nighttime use and users with heightened sensory sensitivity.

Features:

* Dark backgrounds
* Reduced screen brightness
* Lower eye strain
* Calm visual appearance
